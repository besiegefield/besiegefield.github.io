💡 使用说明

  • 输入你的 OpenAI API 密钥（支持 GPT-4）
  • 【可选】如果使用代理或第三方 API，填写 API Base 地址
  • 在下方输入框描述你想要的机器（例如：一辆简单的四轮车）
  • 点击"生成机器"按钮，AI 将生成机器的 JSON 数据
  • 系统自动将 JSON 转换为 XML 格式的 .bsg 文件内容
  • 复制或下载 XML 内容，保存为 .bsg 文件导入 Besiege
💡 留空则使用默认地址。如果使用代理或第三方兼容 API，请填写完整的 Base URL
